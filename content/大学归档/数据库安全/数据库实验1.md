# **数据库安全**-实验1

## 要求

1. 熟悉加密和解密的概念和算法，可以通过使用工具编写程序加密/解密信息。

2. 使用Linux系统的openssl命令行工具。

## 前言

由于目前使用的电脑上面没有虚拟机，我也懒得去装Linux，所以选择另外一个方案：Win32OpenSSL。

这里提供一个轻量版本：

https://wwc.lanzoub.com/ipM2x04ilqvg

（注意：如果自己去官网下载的话，不能使用3.X版本，似乎3.x版本硬编码了一个不应该的数值，导致它完全不可用，切记切记）

另：后面的实验若无注明，则全部使用该软件进行。

## 加解密测试

在解压后的文件目录新建文件夹shiyan1:

![image-20220509004922169](https://pic.firehomework.top/2025/03/7f5b7d1a4240068e113043eccc1ff9d4.webp)

打开并新建一个文件：

![image-20220509004915537](https://pic.firehomework.top/2025/03/43eb281f45adb52b74f118be495f8f13.webp)

修改文件内容：

![image-20220509004909683](https://pic.firehomework.top/2025/03/c74db55adbb63facda01f0e852a7af81.webp)

回到刚刚的框内，输入cd shiyan1进入shiyan1目录：

![image-20220509004903621](https://pic.firehomework.top/2025/03/d66567c857754651c51237e3fa045a20.webp)

我们使用 **openssl enc xxx -e **来加密文件：

```bash
openssl enc -aes-128-cbc -e -in agoodfile.txt -out abadfile.txt
```

（复制后右键就可以直接粘贴进入。）

![image-20220509004857716](https://pic.firehomework.top/2025/03/22cdd36d3a04684750cf5338a285e16a.webp)

要求输入密码，注意输入不会显示。

![image-20220509004844688](https://pic.firehomework.top/2025/03/38663041aa14d1c2be82a04ce75260e4.webp)

加密完毕，查看文件夹内：

![image-20220509004955562](https://pic.firehomework.top/2025/03/864d8de3e55ecdfa2e09a6506a866890.webp)

多出了加密文件，说明文件被加密。

之后使用解密进行解密：

```bash
openssl enc -aes-128-cbc -d -in abadfile.txt -out anothergood.txt
```

![image-20220509005041167](https://pic.firehomework.top/2025/03/8cafe60243545e1a031ad4d77f3818a2.webp)

输入刚才的密码

![image-20220509005105854](https://pic.firehomework.top/2025/03/3c652abf3eb042b541bcd9434e71ea57.webp)

解密成功。
